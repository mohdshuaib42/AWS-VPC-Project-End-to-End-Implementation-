# AWS-VPC-Project-End-to-End-Implementation-
🚀𝗔𝗪𝗦 𝗩𝗣𝗖 𝗣𝗿𝗼𝗷𝗲𝗰𝘁 𝗲𝗻𝗱 𝘁𝗼 𝗲𝗻𝗱 𝗶𝗺𝗽𝗹𝗲𝗺𝗲𝗻𝘁𝗮𝘁𝗶𝗼𝗻🚀
Demonstrated how to create a VPC that we can use for servers in a production environment.

To improve resiliency,deployed the servers in two Availability Zones, by using an Auto Scaling group and an Application Load Balancer. For additional security,deployed the servers in private subnets. The servers receive requests through the load balancer. The servers can connect to the internet by using a NAT gateway. To improve resiliency,deployed the NAT gateway in both Availability Zones.
